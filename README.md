# DelayPlugin

A delay plug in built by Sam Smith - sdsmit@umich.edu

## Usage 

Build with the projucer in xcode/vscode. Load in DAW or run as stand alone app. 

## Features

Delay time, Feedback amount, Delay gain, and Soft Clipping Threshold.
